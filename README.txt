Please remeber to set the following configuratoins: 
1. DB_URL
You can do this by adding the  MONGO_DB_URL=[url] configuration to the .env file 
or set it as an environment variable as MONGO_DB_URL=[url]

2. JWT_PRIVATE_KEY
Add JWT_PRIVATE_KEY=[private key] to the .env file or set as environment as app_jwtPrivateKey=[private key]


Enjoy :)
